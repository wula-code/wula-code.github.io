vim语法

Vim

  • 末行模式

    • set nu - 显示行号
    • set ts=4 - 设置制表键的空格数
    • set expandtab - 将制表键扩展为空格
    • set autoindent - 自动缩进
    • set ruler：显示光标位置
    • syntax on / off：打开 / 关闭语法高亮

  • 命令模式

    1. ZZ - 保存退出

    2. i / I / a / A/ o / O - 进入编辑模式

    3. 移动光标

      1. h j k l ： 左 / 下 / 上 / 右
      2. gg / G / 100G ：开始 / 结尾
      3. 0 /$：语句前 / 后
      4. w / e：单词的开始 / 结尾
      5. ctrl + f / b / e / y：向后一页 / 向前一页 / 向后一行 / 向前一行

    4. 内容编辑

      1. 删除 ：
        1. dd ：删除整行
        2. d$：从光标位置到行尾
        3. d0：从光标位置到行首
        4. dw：删除光标所在的单词
        5. %d：删除所有内容
      2. 撤销：u
      3. 恢复：ctrl + r
      4. 执行上一次命令：.
      5. 复制粘贴：
        1. yy / p
        2. v -- Visual -- 移动光标选中内容 -- y -- p： 进入可视模式，选中复制内容，粘贴

    5. 文件权限

      1. r ：读权限
      2. w ：写权限
      3. x ：执行权限
      4. 修改文件权限：
        1. chmod u + / -x 文件 ：给文件所有者添加/删除执行权限
        2. chmod g + / - x 文件：给同组用户添加/删除执行权限
        3. chmod o + / - r文件：给其他用户添加/删除执行权限

      rw- r-- r--

      文件所有者 同组用户 其他用户
